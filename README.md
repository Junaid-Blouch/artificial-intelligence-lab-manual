## 📚 Overview

This repository contains a complete lab manual for the **Artificial Intelligence** course.  
It covers:

- Python programming fundamentals (variables, conditionals, loops, functions, data structures)
- Data handling & visualization with **NumPy**, **Pandas**, **Matplotlib**, **Seaborn**
- Classical AI algorithms: **BFS** (friend recommendation), **DFS** (web crawling), **A*** (pathfinding), **Hill Climbing** (optimization)
- Dataset exploration (Kaggle, data types, collection methods)
- Supervised learning: **Regression** (predictive modeling) & **Classification** (Logistic Regression on Zoo and Iris datasets)
- Chatbot development (rule‑based + Groq API)
- **Final Project:** Offline **Text‑to‑Speech web application** (Flask + pyttsx3)

All experiments include objectives, theory, algorithms, code screenshots, and sample outputs.
